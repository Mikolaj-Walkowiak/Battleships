BattleShipGame

This is a single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling

After clicking the button below a game of battleship will be simulated between two bots. One of the bots plays 100% randomly, the other one begins its game by going diagonaly from top to bottom and back. You will be shown the final board state althought playtrouh history is currently being made possible to be vieved!
